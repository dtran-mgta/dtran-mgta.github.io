HyperText Markup Language (HTML)

HTML represents the structure of your content. It defines each part of your content, such header, paragraph, or image so that it may be styled by CSS and read by your browser. A common misconception about HTML is that it is a programming language. Since HTML is a markup language, it is much easier to learn, as it only controls the presentation and identification of data. Programming languages are used to manipulate and transform data, like executing a math problem. Javascript, not to be confused with another programming language called Java, is the programming language used with the web. Here is an example of a basic HTML template used for any website. 


      <!DOCTYPE html>
      <html>
        <head>
          <title></title>
        </head>
        <body>

        </body>
      </html>

Elements

At the top, we declare what type of document is being written. The red text, are the name of elements. Elements are what define the structure of your page and consist of two tags, an opening tag and a closing tag, and the content. Contained within your element would be your content.

Tags

Tags are indicated by the angled brackets surrounding the red text. An opening tag begins the element with the name of the element surrounded by angled brackets. The closing tag ends the element with a forward slash before the name of the element.

Attributes

Attributes are additional information about an element. For example, providing the file path to an image in an img element or a path to a link. They are always apart of the opening tag.

Classes

Classes provide a way to add common styles across multiple different elements by including a user defined class name to the element
