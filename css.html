Cascading Style Sheet (CSS)

CSS is responsible for adding the look and feel of a site, such as adding color, fonts or spacing.


      <!DOCTYPE html>
      <html>
        <head>
          <title></title>
          <style>
            CSS HERE
          </style>
        </head>
        <body>

        </body>
      </html>

CSS can be added in a few ways. For a majority of the course, we added styles into a style element contained within the head element. By the end of the course, when our CSS got longer, we separated our CSS from our HTML by making a separate styles.css file and linking to that file in our HTML


      <!DOCTYPE html>
      <html>
        <head>
          <title></title>
           <link rel="stylesheet" href="styles.css">
        </head>
        <body>

        </body>
      </html>
