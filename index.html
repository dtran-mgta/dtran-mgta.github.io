Software

The tools used in the class are all available online for free.

Brackets [http://brackets.io]

Using a text editor specific to coding is not required but very helpful as it can do things like highlight your code so your eyes can follow your code or autocomplete elements. Brackets is our text editor of choice for the course as it allows a quick and easy option to view changes to your code with its live preview tool and color picker for choosing color without needing to know the code value for a color.

GitHub Desktop [https://desktop.github.com]

Git is version control system used for keeping track of changes in a directory. The records, amongst other data including the files themselves, are kept in what's called a repository. GitHub is a service that provides an online host for repositories. Finally, GitHub Desktop is the desktop application we use to interact with GitHub services and logging changes. Lessons in Git and GitHub are outside the scope of the course and aren't required for web development, but using these technologies, we found that it was the best way for the students not to loose their work, not rely on remembering to bring a flashdrive, and most importantly, host their website for free. Students were given the basic skills with GitHub to get their site up and running.
